# Statcounter 各國家數據文件

## 📋 概述

本目錄包含 38 個國家/地區的獨立數據文件，每個文件都：
- ✅ 包含完整的國家數據
- ✅ 明確標註數據來源
- ✅ 適合在 Notebook LM 中單獨使用
- ✅ 支持多國家比較分析

## 📁 文件說明

### 主要文件

- **國家索引.txt** - 所有國家的完整列表和文件對應關係
- **快速參考.txt** - 快速查找卡片，按地區分類
- **Notebook_LM_單一國家使用指南.md** - 詳細使用說明
- **Statcounter_[國家名稱]_數據報告.txt** - 各國家的數據文件（共38個）

### 文件命名規則

```
Statcounter_[國家名稱]_數據報告.txt
```

例如：
- `Statcounter_台灣_數據報告.txt`
- `Statcounter_中國_數據報告.txt`
- `Statcounter_美國_數據報告.txt`

## 🚀 快速開始

### 1. 查找國家文件

打開 **快速參考.txt** 或 **國家索引.txt**，找到你要查詢的國家。

### 2. 在 Notebook LM 中使用

1. 打開 [Notebook LM](https://notebooklm.google.com/)
2. 點擊「新增來源」→「上傳文件」
3. 選擇對應國家的 `.txt` 文件
4. 等待處理完成
5. 開始提問！

### 3. 提問範例

**單一國家：**
- "台灣最流行的瀏覽器是什麼？"
- "中國的作業系統使用情況如何？"

**多國家比較：**
- "比較台灣、中國和美國的瀏覽器市佔率"
- "哪個國家的 Chrome 使用率最高？"

## 📊 數據來源標註

每個文件都包含完整的來源信息：

```
數據來源：Statcounter Global Stats
官方網站：https://gs.statcounter.com/
統計範圍：[國家名稱] 地區的網頁瀏覽統計
數據收集方式：通過 Statcounter 追蹤代碼收集的匿名統計數據
```

**Notebook LM 會自動引用這些來源信息！**

當你提問時，Notebook LM 的回答會包含：
- 數據來源說明
- 來源網站連結
- 數據收集方式
- 數據代表性說明

## 💡 使用場景

### 場景 1：單一國家深度分析
- 上傳一個國家文件
- 深入了解該國的設備使用習慣
- 分析時間趨勢變化

### 場景 2：多國家比較
- 上傳 3-5 個國家文件
- 進行跨國比較分析
- 找出地區差異

### 場景 3：特定主題研究
- 上傳相關國家文件
- 針對特定主題（如瀏覽器、作業系統）進行分析
- 比較不同地區的偏好

## ⚠️ 注意事項

1. **文件大小**
   - 不同國家的文件大小不同
   - 較大國家（如歐洲地區、中國）的文件可能較大
   - Notebook LM 可能需要一些處理時間

2. **數據時效性**
   - 數據範圍：2008-07 至 2026-01
   - 建議定期更新數據

3. **數據代表性**
   - 數據僅代表使用 Statcounter 追蹤的網站
   - 不代表該地區所有網站的訪問情況

4. **Notebook LM 限制**
   - 建議一次上傳 3-5 個國家文件
   - 如果文件太大，可能需要等待處理
